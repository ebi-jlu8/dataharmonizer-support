# Introduction
dataharmonizer-support supports the generation of yaml and json files needed to create templates used in DataHarmonizer. The scripts are specifically designed to generate templates for registration of sample data into ENA.

```
# script takes an <input_tsv_file> and generates a schema.yaml and export.js file for DataHarmonizer.

python scripts/dataharmonizer-support.py --help

usage: dataharmonizer-support.py --input_tsv <input_tsv_file> --result_directory <result_directory>

The dataharmonizer-support script is designed to generate the primary json and yaml files that the DataHarmonizer program needs to generate templates for harmonizing data to a specific format. Harmonized data can then be
used to submit data to specific data archived.

optional arguments:
  -h, --help            show this help message and exit
  --input_tsv INPUT_TSV, -i INPUT_TSV
                        input excel file with all relavent information for generating either the schema.yaml or export.js file
  --result_directory RESULT_DIRECTORY, -r RESULT_DIRECTORY
                        directory to store generated schema.yaml and export.js files.

The export.js and schema.yaml files generated by dataharmonizer-support can be used directly by dataharmonizer to generate templates for entering and harmonizing the data.

```
